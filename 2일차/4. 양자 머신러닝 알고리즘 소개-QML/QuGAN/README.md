# Quantum GAN (QuGAN) 훈련 프로젝트

프로젝트는 양자 생성적 적대 신경망(Quantum Generative Adversarial Network, QuGAN)을 구현하고 훈련하는 코드를 포함합니다. 고전적인 데이터를 양자 상태로 인코딩하여 양자 회로를 통해 데이터를 생성하고, 판별자가 이를 평가하며 학습하는 과정을 시뮬레이션합니다.

## 프로젝트 구조

핵심 기능은 다음 Python 모듈에 구현되어 있습니다:

* **`circuit_func.py`**: QuGAN의 양자 회로 구성 요소(양자 게이트 레이어, 데이터 인코딩, 스왑 테스트), 손실 함수, 파라미터 초기화 등 핵심 양자 논리를 정의합니다.
* **`pca_func.py`**: 이미지 데이터의 차원 축소(PCA)와 양자 회로 입력에 적합한 스케일링, 그리고 역변환을 통한 이미지 복원 기능을 제공합니다.
* **`util_func.py`**: 훈련 설정 관리(YAML), 훈련 과정 시각화(분포 및 생성 이미지 플롯), 훈련 지표 및 모델 파라미터 로깅 등 전반적인 유틸리티 기능을 담당합니다.

## 핵심 모듈의 역할 및 주요 기능

* **`circuit_func.py`**:
    * **양자 회로 빌더**: `training_circuit` 클래스는 판별자가 가짜 데이터(`disc_fake_train`)와 실제 데이터(`disc_real_train`)를 구분하도록 학습하는 데 필요한 양자 회로를 구성합니다. 이 회로들은 `traditional_learning_layer`를 통해 학습 가능한 게이트 레이어를 적용합니다.
    * **데이터 인코딩**: `data_loading_circuit`을 통해 고전적 데이터를 양자 큐비트 상태로 인코딩합니다.
    * **유사도 측정**: `swap_test`를 사용하여 두 양자 상태의 유사도를 측정하고, 이는 판별자의 손실 계산에 활용됩니다.
    * **기울기 계산**: `_apply_parameter_shift` 헬퍼 함수와 파라미터 이동 규칙을 통해 양자 회로 파라미터의 기울기를 추정합니다.
    * **손실 함수**: 판별자와 생성자를 위한 별도의 `cost_function` 및 `generator_cost_function`을 정의합니다.
* **`pca_func.py`**:
    * **데이터 스케일링 및 역스케일링**: `save_descaling_params`를 통해 PCA 데이터를 `[0, 1]` 범위로 정규화하고, `descaling_points`를 통해 원본 스케일로 역변환하여 이미지 복원을 가능하게 합니다.
 
* **`util_func.py`**:
    * **설정 관리**: `load_config_yaml` 및 `save_config_yaml`을 통해 YAML 파일로 훈련 설정을 간편하게 관리합니다.
    * **시각화**: `plot_kde_scatter` 함수를 통해 생성된 데이터 분포(KDE)와 실제 데이터 분포(산점도)를 시각적으로 비교하는 플롯을 생성합니다. `plot_generated_images` 함수는 훈련 과정에서 생성된 이미지 샘플을 플롯하고 저장합니다.
    * **훈련 로깅**: `save_variables` 함수는 각 에포크의 훈련 지표(손실, KL 발산)와 모델 파라미터를 파일로 기록합니다.

## QuGAN (Quantum Generative Adversarial Network) 설명

GAN은 **생성자(Generator)**와 **판별자(Discriminator)**라는 두 개의 신경망이 서로 경쟁하며 학습하는 프레임워크입니다.

* **생성자**: 실제 데이터와 유사한 새로운 데이터(예: 이미지)를 생성하는 것을 목표로 합니다.
* **판별자**: 입력된 데이터가 실제인지 생성자가 만든 가짜인지 구분하는 것을 목표로 합니다.

**QuGAN**은 이러한 GAN의 개념을 양자 컴퓨팅에 적용한 것입니다.

* **양자 생성자**: 양자 회로를 사용하여 무작위 입력으로부터 양자 상태(데이터)를 생성합니다. 이 프로젝트에서는 `training_circuit.disc_fake_train` 함수의 일부 (`traditional_learning_layer`의 후반부)가 생성자 역할을 수행합니다.
* **양자 판별자**: 역시 양자 회로를 사용하여 입력된 양자 상태가 실제 데이터에서 온 것인지, 아니면 양자 생성자가 만든 가짜 상태인지 판별합니다. 이 프로젝트에서는 `training_circuit.disc_real_train`과 `disc_fake_train`의 전반부가 판별자 역할을 합니다.
* **핵심 메커니즘**:
    * **스왑 테스트 (Swap Test)**: 판별자는 주로 스왑 테스트를 통해 생성된 양자 상태와 실제 양자 상태 또는 기준 상태 간의 유사도를 측정하여 판별합니다.
    * **파라미터 이동 규칙 (Parameter Shift Rule)**: 양자 회로에서 학습 가능한 파라미터에 대한 기울기를 계산하기 위해 사용됩니다. 이는 고전적 신경망의 역전파(backpropagation)와 유사한 역할을 합니다.

QuGAN은 양자 컴퓨팅의 이점(예: 양자 병렬성, 복잡한 양자 상태 표현 능력)을 활용하여 고전적 GAN으로는 어려운 특정 유형의 데이터 생성이나 분포 학습에 잠재력을 가집니다.

## 훈련 흐름 (개념적)

QuGAN의 훈련은 에포크(Epoch)별로 반복되며, 각 에포크에서 판별자와 생성자가 번갈아 훈련됩니다.

1.  **데이터 준비**: PCA 변환 및 스케일링된 실제 데이터가 준비됩니다.
2.  **판별자 훈련 단계**:
    * **실제 데이터 판별**: 판별자는 실제 데이터를 입력받아 '실제'라고 올바르게 판단하도록 훈련됩니다.
    * **가짜 데이터 판별**: 생성자의 현재 상태로 가짜 데이터를 만들고, 판별자는 이 가짜 데이터를 '가짜'라고 올바르게 판단하도록 훈련됩니다.
    * 이 과정에서 `cost_function`과 파라미터 이동 규칙을 사용하여 판별자의 파라미터를 업데이트합니다.
3.  **생성자 훈련 단계**:
    * 생성자는 새로운 가짜 데이터를 생성하고, 판별자가 이 데이터를 '실제'라고 오인하도록(즉, 판별자를 속이도록) 훈련됩니다.
    * `generator_cost_function`과 파라미터 이동 규칙을 사용하여 생성자의 파라미터를 업데이트합니다.
4.  **시각화 및 로깅**: 일정 에포크마다 생성된 데이터의 분포 플롯, 생성된 이미지 샘플 등을 저장하여 훈련 진행 상황을 모니터링합니다.

## 핵심 개념

* **파라미터 이동 규칙**: 양자 회로에서 기울기(gradient)를 효율적으로 계산하는 방법입니다.
* **스왑 테스트**: 두 양자 상태의 유사도를 측정하는 양자 알고리즘입니다.
* **Ry 게이트**: 큐비트를 Y축 중심으로 회전시키는 기본 양자 게이트로, 데이터 인코딩 및 학습 가능한 레이어에 활용됩니다.
* **시뮬레이터 선택**:
    * **`AerSimulator(method='statevector')`**: 정확한 양자 상태를 직접 계산하여 훈련이 가능합니다 (노이즈 없는 이상적인 시뮬레이션).
    * **`AerSimulator()` (기본값)**: 샷 기반 시뮬레이션을 수행하여 실제 양자 장치의 측정 통계를 모방합니다.
